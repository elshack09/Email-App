# Email App

Email App using Mongoose, Express, React, and Node with OAuth 2.0. Utilized Stripe Checkout API to process payments and Google OAuth to authorize user login. 

# Tech Stack

## Front End
- Materialize CSS
- React 
- Redux
- Stripe Checkout

## Back End
- Node
- Express
- Passport
- Google OAuth 2.0
- NPM
- Nodemon
- SendGrid: Emaill Delivery Service
